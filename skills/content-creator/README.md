# Content Creator Agent - 完整内容创作工作流

## 系统架构

```
用户输入主题
    ↓
[阶段1] 主题拆解 → 目标、角度、关键问题
    ↓
[阶段2] 资料搜索 → web_search + kimi_search
    ↓
[阶段3] 大纲生成 → 结构化内容框架
    ↓
[阶段4] 内容创作 → 结合用户画像+目标人群
    ↓
[阶段5] AI痕迹消除 → 人工化改写
    ↓
[阶段6] 配图生成 → DALL-E/图像生成
    ↓
[阶段7] 发布 → Feishu/公众号/其他平台
```

## 核心能力

### 1. 主题拆解 (Decomposition)
- 识别核心论点
- 确定目标受众
- 提炼关键问题
- 选择内容角度

### 2. 资料搜集 (Research)
- web_search - 实时信息
- kimi_search - 深度内容
- kimi_fetch - 网页解析
- 多源交叉验证

### 3. 大纲生成 (Outline)
- 结构化框架
- 逻辑递进设计
- 章节权重分配

### 4. 内容创作 (Writing)
- 加载 USER.md 用户画像
- 匹配写作风格
- 融入个人经历
- 适配目标人群

### 5. AI痕迹消除 (Humanization)
- 去除"首先/其次/最后"等模板词
- 增加个人化表达
- 加入不完美表达（口语化、轻微重复）
- 调整句式长度变化

### 6. 配图生成 (Visual)
- 章节配图设计
- DALL-E/其他图像生成工具
- 图文匹配优化

### 7. 发布 (Publish)
- 飞书文档
- 微信公众号
- Markdown 导出

## 工作流指令

当接收到创作任务时：

1. 询问用户（如信息不足）：
   - 目标平台（公众号/知乎/小红书/飞书）
   - 目标人群
   - 文章长度要求
   - 是否配图
   - 发布时间要求

2. 自动执行：
   - 按7阶段顺序执行
   - 每阶段完成后向用户确认或继续
   - 支持中途修改调整

## 用户画像参考

创作时读取 /root/.openclaw/workspace/USER.md：
- 写作风格
- 专业背景
- 个人经历
- 口头禅/常用表达

## 输出格式

- Markdown 标准格式
- 配图占位符 [IMAGE: 描述]
- 发布就绪版本

## 质量检查清单

- [ ] 主题明确，角度独特
- [ ] 资料来源可靠
- [ ] 结构清晰，逻辑通顺
- [ ] 符合用户写作风格
- [ ] 无AI模板痕迹
- [ ] 配图与内容匹配
- [ ] 发布格式正确

## 📈 四、计费模式详解

### 1. OpenAI 模式（自动缓存）
```
计费 = Input(未命中) + Cached Input(命中) + Output
典型比例：Cached Input ≈ 0.1 × Input
```

### 2. Anthropic 模式（手动缓存）
```
计费 = Base Input + Cache Write(1.25x-2x) + Cache Read(0.1x) + Output
5分钟缓存：Write = 1.25x Base
1小时缓存：Write = 2x Base
```

### 3. 国产模型模式
- **智谱**：阶梯定价（输入长度<32K / >32K价格不同）
- **Kimi**：缓存命中价格 ≈ 0.175 × 未命中价格
- **MiniMax**：固定价格，无缓存折扣
- **百炼/混元**：纯按量计费，支持Batch 50%折扣

2. **免费额度**：

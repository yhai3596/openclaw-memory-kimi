## 任务分发规则

### 自动识别关键词

**技术专家 (@tech)**
- 安装、配置、部署、代码、脚本、API
- Scrapling、爬虫、自动化、系统
- 报错、故障、调试、优化

**研究分析师 (@research)**
- 政策、法规、市场、行业、报告
- 泰国、BOI、投资、调研
- 数据、分析、整理、搜索

**AI培训顾问 (@ai)**
- 课程、培训、客户、服务
- 定价、产品、商业模式
- AI落地、企业咨询、方案

**内容创作专家 (@content)**
- 写文章、创作、文案、公众号
- 内容策划、主题、大纲
- 发布、排版、配图
- 去除AI痕迹、人工化改写

### 分发逻辑

```
用户输入
    ↓
主代理分析任务类型
    ↓
匹配子代理 → 转发任务
    ↓
子代理执行 → 返回结果
    ↓
主代理汇总 → 回复用户
```

## 使用方式

用户无需指定，主代理自动判断并分发。
如用户想指定：说 "让技术专家处理" 或 "@tech"

## 优势

1. 上下文隔离 - 各代理独立记忆，互不污染
2. 专业深度 - 每个代理专注领域，回复更精准
3. 并行处理 - 可同时执行多个任务
4. 成本优化 - 简单任务用轻量模式，复杂任务用深度思考

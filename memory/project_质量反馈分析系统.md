# 质量反馈智能分析系统 - 项目记录

## 项目概览
- **项目名称**: 质量反馈智能分析系统
- **目标用户**: 质量经理（空调硬件产品，美国市场）
- **核心目标**: 每周质量分析时间从 N 小时 → 30 分钟
- **数据语言**: 纯英文
- **数据来源**: Zoho Desk Tickets

## 关键信息

### 样本数据
- **历史样本**: 2000+ 条已分类数据
- **字段**: Subject + Description + Ticket Type
- **机型字段**: Condenser Model Number
- **格式**: Zoho Desk 导出 Excel

### 分类体系
| 一级分类 | 二级分类示例 |
|---------|-------------|
| 非质量问题 | 经销商咨询、销售咨询、保修注册、税务咨询 |
| 功能失效 | 不制冷、不制热、无法启动 |
| 漏水 | 室内机漏水、室外机漏水 |
| 噪音 | 外机噪音、内机噪音 |
| 安装问题 | 安装不当、管道问题 |
| 显示故障 | 错误代码 |
| 外观瑕疵 | 外观损坏 |

### 技术方案
- **分类模型**: 可配置（BAAI/bge-m3 推荐）
- **界面**: Streamlit
- **简报推送**: 飞书 Webhook，每周四 8:30
- **存储**: SQLite

### 业务规则
- **QC 触发**: 同类问题（问题现象+机型）≥2 起
- **置信度阈值**: <0.75 需人工复核
- **评分公式**: 严重度 × 次数 × 趋势系数

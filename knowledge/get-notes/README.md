# Get 笔记知识库 - 陈天式方案

## 理念
参考 sky 陈天的方法论：**把数据拉回本地，构建个人数据资产**

## 目录结构

```
knowledge/get-notes/
├── import/          # 放入 Get 笔记导出的文件
├── raw/             # 原始备份
├── by-date/         # 按日期归档
├── by-topic/        # 按主题分类（AI 自动）
├── insights/        # 提取的洞察、金句、案例
└── persona.md       # 个人知识画像
```

## 使用流程

### 第一步：导出 Get 笔记
1. 打开 Get 笔记 App
2. 选择要导出的内容
3. 导出为 Markdown 或 TXT
4. 将文件发送到飞书，我保存到 `import/` 目录

### 第二步：自动导入
我运行同步脚本，自动：
- 备份原始文件
- 按日期归档
- 生成导入报告

### 第三步：AI 分析（我执行）
- 自动分类（会议/灵感/客户/待办）
- 提取金句和核心观点
- 识别可写成文章的选题
- 更新个人知识画像

### 第四步：内容生成
基于分析结果：
- 生成文章大纲
- 输出符合你风格的初稿
- 提供选题建议

## 优势

| 云端方案 | 本地方案（陈天式） |
|---------|------------------|
| 依赖 API，可能失效 | 文件在手，永久可用 |
| 隐私风险 | 数据完全掌控 |
| 功能受限 | 可选择最佳 AI 模型分析 |
| 无法深度定制 | 完全按你需求定制 |
